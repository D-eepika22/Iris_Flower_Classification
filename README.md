# Iris Flower Classification #

## Overview
This project uses Machine Learning to classify Iris flowers into three species:
Setosa
Versicolor
Virginica

## Features
Sepal Length
Sepal Width
Petal Length
Petal Width

## Technologies Used
Python
NumPy
Pandas
Scikit-Learn

## Model
Random Forest Classifier

## Accuracy
100%

## Run Project
python iris_model.py
